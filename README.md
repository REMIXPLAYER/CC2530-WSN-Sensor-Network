# CC2530 无线传感器网络

基于TI CC2530芯片的无线传感器网络项目，实现温湿度、光照数据的采集与无线传输。

## 项目结构

```
├── 4.1-P2P-DHT11/          # DHT11温湿度传感器节点
├── 4.1-P2P-Photoresistance/ # 光敏电阻传感器节点
└── 4.1-P2P-Recevier/        # 数据接收融合节点
```

## 硬件平台

- **芯片**: TI CC2530 (8051内核, 2.4GHz RF)
- **开发环境**: IAR Embedded Workbench for 8051
- **通信协议**: BasicRF (2.4GHz)
- **信道**: 24
- **PAN ID**: 0x1410

## 节点说明

| 节点 | 地址 | 功能 |
|------|------|------|
| DHT11发送节点 | 0x4103 | 采集温湿度数据，通过RF发送 |
| 光敏电阻发送节点 | 0x4101 | 采集光照强度数据，通过RF发送 |
| 接收融合节点 | 0x4102 | 接收并融合所有传感器数据 |

## 通信协议

- **频率**: 2.4GHz ISM频段
- **调制方式**: O-QPSK
- **数据速率**: 250kbps
- **CSMA-CA**: 实现载波侦听多路访问冲突避免

## 数据格式

### DHT11节点发送数据
```
{Humidity=xx, Temp=xx}
```

### 光敏电阻节点发送数据
```
{A0=xx}
```

### 接收节点融合输出
```
{Temp=xx, Humidity=xx, Light=xx, D7=xx}
```

## 配置说明

每个节点通过 `NODE_TYPE` 宏定义角色：
- `NODE_TYPE = 1`: 发送节点
- `NODE_TYPE = 0`: 接收节点

## 使用方法

1. 使用IAR Embedded Workbench打开对应的工程文件(.eww)
2. 根据硬件连接修改引脚定义
3. 编译并烧录到CC2530开发板
4. 通过串口(38400波特率)查看输出数据

## 目录说明

- `*.c / *.h`: 源代码文件
- `Release/Exe/`: 编译输出的可执行文件(.hex, .d51)
- `settings/`: IAR工程配置文件
- `*.ew*`: IAR工程文件
